# ML-powered financial analytics system with prediction + insights

## Project Description
The Expense Tracker ML application is designed to help users manage their expenses effectively through machine learning algorithms. This application allows users to track their daily expenses, categorize them, and monitor their spending habits trend over time.

### Key Features:
- **User Authentication:** Users can securely sign up and log in to maintain their individual expense records.
- **Expense Tracking:** Users can add, edit, or delete expenses with ease and categorize them for better analytics.
- **Data Visualization:** The application provides graphs and charts to visualize spending habits and trends over time.
- **Machine Learning Analysis:** Utilizes machine learning algorithms to predict future expenses based on historical data, helping users to plan their finances better.
- **Alerts and Notifications:** Users receive notifications when they exceed their monthly budget in specific categories.

### Technologies Used:
- Python
- Flask/Django (for the web framework)
- SQLite/PostgreSQL (for the database)
- Machine Learning Libraries (such as scikit-learn)
- Frontend Technologies (React/Angular)

### Installation Instructions:
1. Clone the repository.
2. Install the required dependencies.
3. Set up the database.
4. Run the application using the provided commands.
