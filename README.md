# Flood Shelter Recommendation System

## Overview
A decision-support system designed to recommend optimal flood shelters based on uncertain environmental conditions using fuzzy logic.

## Problem Statement
During floods, selecting the safest shelter is difficult due to uncertain and dynamic environmental factors such as water level, distance, and safety conditions.

## Solution
This system uses fuzzy logic to handle uncertainty and provide reliable shelter recommendations by analyzing multiple input parameters.

## Key Features
- Handles uncertain real-world data using fuzzy logic
- Provides data-driven shelter recommendations
- REST API-based backend for real-time processing
- Multi-user support

## Technologies Used
- Python
- Flask
- SQLite
- scikit-fuzzy

## System Workflow
1. Input environmental parameters
2. Apply fuzzy logic rules
3. Process data through inference engine
4. Generate optimal shelter recommendation

## Use Case
Can be used by disaster management authorities to support decision-making during floods.
